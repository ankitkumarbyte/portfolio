## 🌟 Features

### 🎭 **Interactive UI/UX**
- **AI-Powered Chat Interface** - Ask questions in natural language to navigate the portfolio
- **Speech Bubble Animations** - Dynamic avatar with conversational speech bubbles and timed messages
- **Smooth Page Transitions** - Seamless navigation between sections with CSS animations
- **Dark/Light Mode Toggle** - Custom theme switcher with persistent localStorage preferences
- **Liquid Distortion Effects** - SVG filter animations for visual appeal

### 📊 **Rich Portfolio Content**
- **Project Showcase** - 10+ detailed projects with background images and GitHub links
- **Skills Visualization** - Animated skills flow with Devicon integration
- **Experience Timeline** - Chronological work history with detailed descriptions
- **Publication Section** - Research papers with interactive PDF previews
- **Personal Narrative Pages** - Dedicated pages for "Traveller Life" and "Entrepreneur Journey"
- **Contact Integration** - Direct email form (Formspree) and social media links

### ⚡ **Performance & Responsiveness**
- **Mobile-First Design** - Fully responsive across all devices (mobile, tablet, desktop)
- **Image Optimization** - Lazy loading with CSS fallback gradients for project cards
- **Fast Loading** - Minimal dependencies and optimized assets
- **Accessibility** - Semantic HTML, keyboard navigation, and proper ARIA labels
